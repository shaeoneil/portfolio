Recent Projects



  • Business Analytics Capstone

    Data Cleanup for clients new ERP system; Acting as a Business Consultant

    The goal of this project was to consolidate and clean 4 data sets for a successful implementation of new ERP system. Together we merged 4 separate data sets using a left join in R . After the data sets were succesfully merged we randata quality reports and created a Tableau Story Board for an exploratory analysis of initial key findings to better understand the data. After we did this did data preparation: imputing numerical missing variables with 0 and categorical with NA, created a “Key” column concatenating the item number and item description for easier identification of items. We also deleted exact duplicates of data found using conditional formatting. Through teamwork we identified similar items in the data by using Fuzzy Text matching in R and confidence intervals to determine how likely these were to be duplicated items. In the end, we successfully cleaned client’s data and presented findings to client using Tableau and Excel with recommendations on how to improve the data cleanup process in the future.

  • Data Science *MSBA Course*

    Bank Customer Churn Problem

    Using an unbalanced dataset from Kaggle using data mining we looked at what people are most likely to leave the bank as customers. In Orange using the CRISP DM process we first did an exploratory analysis on our data. Then we prepared the data for modeling. Also in Orange we then trained random forests,logistic regression, neural networks, SVM, AdaBoost, and decision tree models to get best parameters to test our data on. Once we found the best parameters based off of the ROC analysis were found we created appropiate visualizations for each model to present our findings of the people most likely to churn from the bank.

    Data Wrangling

    College Salary Analysis

    Took raw data from Kaggle which compares salaries by college, region, and major. Then web scraped data from Wikipedia using RStudio on college Power 5 conferences. We then integrated these two datasets by a vertical merger within R. After data was prepped we performed an analysis to examine which colleges and conferences have the best salaries by using Q-Q plots, bar charts, t-tests, and ANOVA tests in RStudio.
